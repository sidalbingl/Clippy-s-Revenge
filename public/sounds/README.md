# Audio Files

This directory contains sound effects for Evil Clippy.

## Files

- `low.mp3` - Soft beep for low severity
- `medium.mp3` - Glitch/static sound for medium severity
- `high.mp3` - Deep distorted bass + glitch for high severity

## Generating Sounds

You can replace these with your own sounds or use online tools:

### Option 1: Use Online Generators
- [Bfxr](https://www.bfxr.net/) - Retro sound effects
- [ChipTone](https://sfbgames.itch.io/chiptone) - 8-bit sounds
- [Audacity](https://www.audacityteam.org/) - Generate tones

### Option 2: Use the Placeholder Sounds
The current files are simple generated tones that work as placeholders.

### Recommended Settings

**Low Severity:**
- Short duration (0.1-0.2s)
- Soft volume
- Simple beep or click

**Medium Severity:**
- Medium duration (0.3-0.5s)
- Glitchy/static sound
- Moderate volume

**High Severity:**
- Longer duration (0.5-1s)
- Deep bass + distortion
- Loud and aggressive

## File Format

- Format: MP3 or WAV
- Sample Rate: 44100 Hz
- Bit Rate: 128 kbps (MP3)
- Mono or Stereo

## Testing

Use the Dev Controls in the app to test each sound level.
